    Opgave 6 - Tweetanalyse

    Hent dette zip-arkiv: trump_clinton_tweets.zip. Opret en ny mappe og udpak filerne til din projektmappe.

    Arkivet indeholder to filer:

    • trump_clinton.json: En JSON-fil med 193.705 tweets, hvor teksten eller brugernavnet indeholder “Trump” eller “Clinton”. Alle tweets er fra 1. september og indsamlet 13. september.
    • trump_clinton_100.json: 100 tilfældigt udvalgte tweets fra ovenstående fil.

    Hvert tweet er et dictionary der indeholder følgende felter: (datatype i parentes)

    • text: Tweetets tekst. (str)
    • user: Brugernavn. (str)
    • fullname: Fulde navn på bruger. (str)
    • replies: Antal svar. (int)
    • likes: Antal likes. (int)
    • retweets: Antal retweets. (int)

    Hvis tweet er en variabel der indeholder et tweet, kan du trække et felt ud med udtrykket tweet["field"]. Følgende eksempel indlæser hele JSON-filen og printer brugernavn som tekst til terminalen:

    import json
# Læs alle tweets ind som en liste.
tweets = json.load(open("trump_clinton_100.json", "r"))

# Løber gennem all tweets og printer brugernavn samt tekst.
# Eksempel:
#   @piersmorgan: Donald Trump being abused for donating $1 million to #HarveyRelief
#   is peak anti-Trump liberal hysteria gone haywire. Get over yourselves.
for tweet in tweets:
	user = tweet["user"] # brugernavn
	text = tweet["text"] # tekst
	print("@" + user + ": " + text) # printer "@user: text"

    Tip: Arbejd først med filen med 100 tweets. Når dit program virker benyt derefter den store fil.

    • Benyt ovenstående kode og test at der udskrives det rigtige antal tweets.
    • Hvor mange tweets indeholder ordet "Trump"?
    • Hvor mange tweets indeholder ordet "Clinton"?
    • Er der flest tweets der indeholder både “corrupt” og "Trump" eller "corrupt" og "Clinton"?
    • Kan du finde det mest likede tweet? Print tweetets tekst, navnet på brugeren og antal likes.
    • Hvad er det gennemsnitlige antal likes over alle tweets? (Bonus: Er gennemsnittet forskelligt fra tweets der indeholder Trump eller Clinton?)
    • Prøv at sammenligne dine tal ved at tegne et søjlediagram. Du kan bruge nedenstående skabelon.
