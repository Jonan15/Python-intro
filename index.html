Forberedelse

Før I møder til første lektion skal I så vidt muligt have installeret python3 samt en teksteditor.

Installation af python:

  1. Download python installationsfilen her. Vælg mellem windows/linux/mac.
  2. Kør den downloadede installationsfil.
  3. Tjek at python er installeret rigtigt ved at køre følgende kommando i en terminal/komandoprompt: 

    python --version

    Her skal python svare tilbage med den version I har installeret, dvs.: 

    Python 3.7.0

    Hvis I har downloaded fra det ovenstående link.

Der er mange valgmuligheder ift. tekstredigeringsprogram, alt efter hvor meget hjælp man vil have. Følgende er anbefalelsesværdige:

  • Sublime: En meget hurtig teksteditor. Giver ingen ekstra hjælp.
  • Visual Studio Code: Hurtig teksteditor, hvor diverse hjælpeudvidelser nemt kan installeres.
  • Pycharm: En lidt tungere editor, men som giver rigtig meget hjælp, blandt andet med indbygget dokumentation.
